# Правила проекта

## Языки

- **Общение с ИИ-ассистентом (Cline):** только на русском языке.
- **Код, комментарии, UI (интерфейс пользователя):** на испанском языке (es).
- **Документация проекта:** на испанском, если не указано иное.

## Соглашения

- HTML-файлы: `lang="es"`, DOCTYPE и атрибуты на испанском.
- Комментарии в коде: на испанском или английском (по ситуации), но НЕ на русском.
- Названия переменных, функций, классов: на английском (camelCase).

## Цель

Разделение языка общения с ИИ (русский) и языка проекта (испанский) для удобства разработки и соответствия испаноязычной аудитории.
