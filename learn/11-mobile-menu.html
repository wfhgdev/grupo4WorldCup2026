📱 Menú Móvil Compartido

📖 Para principiantes:
  • IIFE — Immediately Invoked Function Expression. Función que se ejecuta inmediatamente. Sirve para aislar variables.
  • Bloqueo de scroll — evitar que la página se desplace mientras el menú está abierto.
  • addEventListener — suscribirse a eventos: clic, pulsación de tecla.
  • classList.toggle — alternar una clase. Si existe la elimina, si no la agrega.

¿Qué es?

Es un overlay de pantalla completa para navegación móvil. No es una página nueva: se muestra sobre la vista actual. Código fuente: [Fuente: ../js/menu.js].

🧱 Mental model: piénsalo como una ventana emergente integrada con la navegación del sitio.

HTML del menú

En cada página hay un div#mobile-menu-overlay oculto con hidden. Ejemplo desde [Fuente: ../index.html]:

<div id="mobile-menu-overlay" class="fixed inset-0 z-[100] hidden">
  <nav>
    <a href="../matches/matches.html">Partidos</a>
    <a href="../groups/groups.html">Grupos</a>
    <a href="../knockout/knockout.html">Eliminatorias</a>
    <a href="../stats/stats.html">Estadísticas</a>
  </nav>
</div>

Los enlaces usan rutas relativas desde subpáginas como learn/ para mantenerse funcionando localmente.

JavaScript (menu.js)

La lógica está dentro de una IIFE para evitar contaminar variables globales.

(function() {
  'use strict';

  function initMobileMenu() {
    const openBtn = document.getElementById('open-menu');
    const closeBtn = document.getElementById('close-menu');
    const menu = document.getElementById('mobile-menu-overlay');

    function toggleMenu(show) {
      menu.classList.toggle('hidden', !show);
      document.body.style.overflow = menu.classList.contains('hidden') ? '' : 'hidden';
    }

    openBtn.addEventListener('click', () => toggleMenu(true));
    closeBtn.addEventListener('click', () => toggleMenu(false));
  }

  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', initMobileMenu);
  } else {
    initMobileMenu();
  }
})();

⚠️ Error común: usar solo stopPropagation() sin bloquear el scroll real del body. En móviles no alcanza.

Body Scroll Lock

Bloqueamos scroll del fondo mientras el menú está abierto.

document.body.style.overflow = 'hidden'; // abrir
document.body.style.overflow = '';        // cerrar

Fuente: [Fuente: ../js/menu.js].

IIFE explicada

IIFE = función que se ejecuta inmediatamente, encapsulando variables internas.

(function() {
  var mensaje = 'Esto está encapsulado';
  console.log(mensaje);
})();

Hoy se usan más los módulos ES, pero sigue siendo útil para scripts pequeños. [MDN — IIFE EN]

📖 For beginners:
  • IIFE — Immediately Invoked Function Expression. Runs as soon as it's defined. Used for variable isolation.
  • Scroll lock — prevent the page from scrolling while the menu is open.
