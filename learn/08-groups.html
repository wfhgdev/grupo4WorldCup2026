¿Qué hace esta página?

Muestra tablas de posiciones calculadas desde los partidos con getGroupMatches(), no desde el endpoint de standings de la API (aunque getStandings() existe en js/api.js, esta página no lo usa — ver Lección 03). Fuentes: [Fuente: ../groups/groups.html] y [Fuente: ../groups/groups.js].

🧱 Por qué calcular: calcular desde partidos te da control sobre columnas, orden y validaciones frente a ajustes del torneo.

calculateTeamStats()

Esta función construye PJ, G, E, P, GF, GC, DIF y PTS por equipo a partir de los partidos del grupo.

const sorted = [...teamsStats].sort((a, b) => {
  if (b.points !== a.points) return b.points - a.points;
  if (b.goalDifference !== a.goalDifference) return b.goalDifference - a.goalDifference;
  return b.goalsFor - a.goalsFor;
});

Fuente: [Fuente: ../groups/groups.js].

⚠️ Honestamente: el criterio de orden implementado es Puntos → Diferencia de goles → Goles a favor. El reglamento oficial de la FIFA añade un desempate por resultado directo (head-to-head) entre los equipos empatados antes de llegar a diferencia de goles general — ese criterio no está implementado aquí porque la API no expone una tabla precalculada con ese dato.

📖 For beginners:
  • reduce() — array method that accumulates a single value from all elements.
  • sort() — array sorting. Takes a comparison function.
  • Ternary operator — condition ? value1 : value2. Shorthand for if-else.
  • Destructuring — let { points, goalsFor } = team; — extract fields from an object.

What it does

Shows standings computed from matches via getGroupMatches(), not from the API's standings endpoint (getStandings() exists but is unused here). Source: [Source: ../groups/groups.js].

Key logic

Sort by points → goal difference → goals for.

⚠️ Honestly: FIFA's official head-to-head tie-break between level teams is not implemented — the API doesn't expose that data pre-computed.
