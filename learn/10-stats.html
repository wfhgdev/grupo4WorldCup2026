Máximos Goleadores

Se consume /competitions/WC/scorers y se muestra avatar inicial, nombre, equipo o nacionalidad y total de goles. Ejemplo desde [Fuente: ../stats/stats.js]:

displayScorers.forEach((s) => {
  const item = document.createElement('li');
  item.innerHTML = `
    <div class="w-8 h-8 rounded-full ... flex items-center justify-center">
      ${s.player.name.charAt(0).toUpperCase()}
    </div>
    <p>${s.player.name}</p>
    <p>${teamName} · ${nationality}</p>
    <span>${s.goals}</span>
  `;
  container.appendChild(item);
});

Consejo: usa siempre charAt(0).toUpperCase() como avatar temporal cuando no tengas imagen.
