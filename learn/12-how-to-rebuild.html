🛠️ Cómo Reconstruir Este Proyecto

📖 Para principiantes:
  • PHP built-in server — php -S localhost:8000 — iniciar el servidor integrado de PHP para pruebas.
  • GitHub Pages — hosting gratuito para sitios estáticos. No soporta PHP.
  • Serverless — funciones en la nube que se ejecutan sin servidor propio. El proxy actual usa una función serverless (api/proxy.js).

Guía paso a paso para crear tu propia versión desde cero. Cada paso referencia archivos reales del proyecto.

Paso 5: Proxy (Serverless)

Crea api/proxy.js como función serverless (para Vercel). Fuente: [Fuente: ../api/proxy.js].

export default async function handler(req, res) {
  const endpoint = req.query.endpoint || '';
  const token = process.env.API_TOKEN; // variable de entorno en Vercel
  const url = 'https://api.football-data.org/v4' + endpoint;

  const response = await fetch(url, {
    headers: { 'X-Auth-Token': token }
  });
  const data = await response.json();
  res.status(response.status).json(data);
}
