  • CSS: estilos personalizados en index.css y clases de Tailwind [Fuente: ../tailwindCustom.js]
  • JavaScript: DOM, eventos, async/await y fetch en [Fuente: ../js/api.js]
  • API: consumo de datos deportivos y proxy en [Fuente: ../api/proxy.js]
  • Datos: cómo se transforman partidos, grupos, eliminatorias y estadísticas
  • i18n: textos en español e inglés desde [Fuente: ../js/i18n/i18n.js] y diccionarios en [Fuente: ../js/i18n/es/]

📖 Para principiantes: ¿qué es cada cosa?

  • HTML – lenguaje de marcado. Son los huesos: los tags (
    ,

    ,

    ) forman el esqueleto de la página.

  • CSS – lenguaje de estilos. Es la ropa y el maquillaje: colores, fuentes, posición en la pantalla.
  • JavaScript – lenguaje de programación. Es el cerebro y los músculos: reacciona a clics, carga datos del servidor.
  • Tailwind CSS – librería de clases CSS ya hechas. En vez de escribir color: red;, pones class="text-red-500" en el HTML.
  • API – contrato entre programas. Como un camarero: tú dices "tráeme el menú" (petición), él va a la cocina y te lo trae (respuesta).
  • CORS – política de seguridad del navegador. El camarero solo puede servir comida de su propia cocina, no del restaurante de al lado.
  • Proxy – intermediario. El camarero va al restaurante de al lado por ti porque a ti no te dejan entrar.
  • localStorage – caja de almacenamiento en el navegador. Como una libreta: los datos siguen ahí aunque cierres la página.
  • JSON – formato de intercambio de datos. Como un formulario: { "nombre": "Brasil", "goles": 2 }.

├── groups/
├── knockout/
└── stats/

Cada carpeta de sección tiene su propio HTML/JS, pero comparten recursos comunes en js/.

Mental model del proyecto

Imagina el sitio como una casa:

  • HTML = la estructura: paredes, puertas y ventanas.
  • CSS = la decoración: colores, luces,mobiliario.
  • JS = la electricidad y automatismos: responde a tus acciones.
  • API = el servicio de delivery que trae información real.
  • Proxy = el recepcionista que evita problemas de identidad.
  • Cache = la nevera: guarda lo último para no pedirlo otra vez.
  • i18n = el sistema de idiomas de la casa.
